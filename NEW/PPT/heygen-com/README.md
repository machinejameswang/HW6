# HeyGen Import Package

Purpose: create a presenter video or avatar narration from the HW6 PowerPoint.

Recommended voice/persona:

- Persona: calm, professional male presenter
- Tone: steady, confident, business-oriented
- Pace: medium-slow
- Language: English narration
- Use case: data science project presentation

Files to use:

- PowerPoint with embedded male narration:
  `../hw6_50_startups_crispdm_20_slides_male_voice_embedded.pptx`
- Voice-over script:
  `../hw6_20_slide_voiceover_script.txt`
- Per-slide audio files:
  `../audio/slide_01_male_narration.wav` through `../audio/slide_20_male_narration.wav`
- HeyGen CSV prompt/script package:
  `heygen_slide_script_package.csv`

Suggested HeyGen workflow:

1. Upload the PPTX or export the slides as images.
2. Choose a stable male presenter avatar.
3. Use the slide-by-slide narration text from the CSV.
4. If using audio upload, attach the matching WAV file for each slide.
5. Keep background minimal and professional.
6. Export as a presentation video.

Important note:

The narration is a learning-project voice-over. Review pronunciation of terms
such as CRISP-DM, Scikit-learn, R&D, and OneHotEncoder before final export.
