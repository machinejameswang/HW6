# Hyperframes Storyboard Package

Purpose: organize the HW6 narrated presentation as a structured video storyboard.

Files:

- `hyperframes_storyboard.json`: slide-by-slide storyboard.
- `../audio/`: male narration WAV files.
- `../hw6_50_startups_crispdm_20_slides_male_voice_embedded.pptx`: PowerPoint with embedded audio.

Recommended visual direction:

- Style: steady business presentation
- Presenter: calm professional male voice
- Motion: slow zooms, light pan transitions, no aggressive animation
- Duration: match each slide to its narration WAV file
- Export: 1080p or higher

Suggested workflow:

1. Import the PPT slides or export each slide as an image.
2. Use each storyboard frame as a scene.
3. Attach the matching `slide_XX_male_narration.wav`.
4. Keep transitions subtle and professional.
5. Render final video.
