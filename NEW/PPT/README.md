# HW6 PowerPoint Package

Main deliverable:

- `hw6_50_startups_crispdm_20_slides_male_voice_embedded.pptx`

This is the 20-slide PowerPoint with embedded stable male narration audio.

Supporting files:

- `hw6_50_startups_crispdm_20_slides_with_notes.pptx`  
  PPT with speaker notes, without embedded audio.
- `audio/`  
  Per-slide WAV narration files generated with `Microsoft David Desktop`.
- `hw6_20_slide_voiceover_script.txt`  
  Full voice-over script.
- `hw6_20_slide_voice_narration_plan.md`  
  Slide-by-slide presentation plan.
- `generate_male_voice_audio.ps1`  
  Regenerates the male WAV narration files.
- `embed_male_voice_in_ppt.py`  
  Embeds the WAV files into the PPTX.
- `make_hw6_ppt.py`  
  Regenerates the 20-slide PPT with speaker notes.
- `heygen-com/`  
  HeyGen import package.
- `hyperframes/`  
  Hyperframes-style storyboard package.

Recommended presentation file:

```text
hw6_50_startups_crispdm_20_slides_male_voice_embedded.pptx
```
